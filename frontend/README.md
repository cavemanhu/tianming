# 天命阁微信小程序 - 前端使用说明

## 📁 文件位置
```
C:\tianming_frontend\
```

## 🔧 微信开发者工具导入步骤

### 1. 下载安装微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 选择 Windows 64 位版本

### 2. 导入项目
1. 打开微信开发者工具
2. 点击「导入项目」
3. 项目目录：`C:\tianming_frontend`
4. AppID：输入你的小程序 AppID（需在微信公众平台注册）
5. 项目名称：`天命阁`

### 3. 配置编译模式
- 编译模式：默认
- JS编译模式：vue3

### 4. 修改后端API地址（如需要）
如果后端部署在不同地址，修改：
```
C:\tianming_frontend\src\utils\request.js
```
将 `BASE_URL` 改为实际后端地址：
```javascript
const BASE_URL = 'http://192.168.3.45:3000'  // 开发环境
// const BASE_URL = 'https://api.tianming.com'  // 生产环境
```

## ⚠️ 注意事项

1. **调试模式**：微信开发者工具需开启「不校验合法域名」（开发阶段）
   - 位置：详情 → 本地配置 → 勾选「不校验合法域名...」

2. **网络要求**：手机和小程序需能访问 `192.168.3.45:3000`
   - 同一局域网内可直连
   - 或需要内网穿透到公网

3. **微信登录**：需要真实的微信 AppID 才能测试登录功能
   - 测试 code 可用，但无法获取真实用户

## 🔑 微信小程序 AppID 获取
1. 登录 https://mp.weixin.qq.com/
2. 进入「开发」→「开发设置」
3. 获取 AppID
